## Installation

- Step 1:clone repository
```bash
git clone https://github.com/ThaoLyyy/react-training
```
- Step 2: Checkout branch
```bash
git checkout branch develop
```
- Step 3:cd folder practice
```bash
cd practice
```
- step 4: 
```bash
npm install
```
- step 5 : run project
```bash
npm start
```
- step 6: Open in browser:
```
~http://localhost:3000~
```
- step 7 : run storybook
```bash
npm run storybook
```
